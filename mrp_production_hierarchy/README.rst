========================
MRP Production Hierarchy
========================

This module extends the functionality of Manufacturing Order
to show a navigable hierarchy.


Usage
=====

To use this module, you need to:

* go to menu Manufacturing - Manufacturing - Production Hierarchy

or

* Run Production Hierarchy wizard from the production order form

.. image:: https://odoo-community.org/website/image/ir.attachment/5784_f2813bd/datas
   :alt: Try me on Runbot
   :target: https://runbot.odoo-community.org/runbot/129/8.0
